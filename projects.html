Music Player

Night Mode

Web application to play your lastfm's favorite tracks using youtube's API. The application allows you to browse through your 500 lastest favorited tracks and play them on demand or on shuffle mode.The application relies on scraping youtube's html and matching video ids to song titles retrieved from LastFM's API. The app is fully tested using RSpec and Capybara and built on Rails 5.1.4.

Inbox

This is a platform that allows users to schedule appointments with the plataform's admin as well as send messages to the admin through an internal messaging system. I developed the platform using Ruby on Rails and integrated the ruby Gem MailBoxer to handle the internal messaging system. I also used the gem Simple_Calender to display an updated calendar to the user with their appointments. For the frontend I used Bootstrap and collaborated with a graphic designer to create a richer user experience.
