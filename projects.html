ScheduleIT

View code | Live Demo

My client needed a platform where his clients could shedule appointments with him, as well as the ability to message him through an internal messaging system. I developed the platform using Ruby on Rails and integrated the ruby Gem MailBoxer to handle internal messaging system as well as the gem Simple_Calender to display an updated calendar to the user. For the frontend I used Bootstrap and collaborated with a graphic designer to create a richer user experience.
